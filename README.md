## Live Demo

[FocusHub Live URL](https://focushub.co.in/)

## Features

- ✅ Task management
- ✅ Goals tracking
- ✅ Focus timer
- ✅ Dashboard for accountability

## Tech Stack

- React, TailwindCSS
- Node.js, Express, MongoDB
